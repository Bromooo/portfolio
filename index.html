Languages I speak:

HTML


HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. I use HTML to describe the structure of a Web page. HTML consists of a series of elements.

CSS


CSS is the acronym of “Cascading Style Sheets”. CSS is a computer language for laying out and structuring web pages (HTML or XML).
I use it for describing the presentation of a document written in HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

Sass


Sass -> Syntactically Awesome Style Sheets Sass is simply CSS with superpowers. Sass is the most mature, stable, and powerful professional grade CSS extension language in the world.
Sass has features that don't exist in CSS yet like variables, nesting, mixins and so on.
I prefer using Sass to CSS.

JavaScript


JavaScript ("JS" for short) is a full-fledged dynamic programming language that can add interactivity to a website.
I use JavaScript to adds interactivity to websites.
